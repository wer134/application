프로젝트 개요

Spotlight Cam은 크로마키의 물리적 제약을 해결하기 위해 개발된 AI 기반 비디오 처리 시스템입니다.

대형 녹색 스크린 없이도 실시간으로 배경을 제거하고 객체를 추적하여, 더 넓은 활동 반경과 낮은 비용으로 고품질 비디오 제작을 가능하게 합니다.

AI 객체 추적

YOLO 객체 인식 알고리즘으로 사람 탐지 및 DeepSORT로 움직임 추적. 라즈베리파이 팬/틸트 모터를 통한 자동 카메라 제어

🎬

실시간 배경 제거

AI를 통한 추적 객체의 배경 제거. 크로마키 없이도 깔끔한 배경 분리 및 다른 배경으로 교체 가능

🎥

비디오 녹화

WebM 형식으로 고품질 녹화. HD/30FPS 초기 목표 성능. 녹화 시간 표시 및 사용자 지정 저장 위치 선택

🎛️

OBS 스타일 UI

다크 테마 인터페이스. 장면(Scenes) 및 소스(Sources) 관리. 실시간 오디오 믹서 및 볼륨 조절

⚙️

설정 기능

카메라/마이크 선택, 해상도 설정, 파일 형식 선택, 저장 위치 설정, 라즈베리파이 연결 설정

🔄

자동 업데이트

GitHub Releases를 통한 자동 업데이트 확인. 사용자 친화적인 업데이트 알림 및 다운로드
